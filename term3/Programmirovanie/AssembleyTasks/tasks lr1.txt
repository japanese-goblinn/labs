В каждом из заданий переменные a, b, c, d определяются в сегменте данных и
имеют размерность слово. Необходимо выполнить над ними заданные арифметические 
и логические операции, а результат поместить в регистр AX. При выполнении 
умножения считаем, что результат вмещается в слово. При выполнении деления 
считаем, что оно целочисленное. Выполнение программы необходимо показать в 
Отладчике.


Варианты заданий:
1) Если a ^ 3 > b ^ 2 то
        Если c * d = a / b то
           Результат = a AND b
        Иначе
           Результат = с
     Иначе
        Результат = с * d + b

2) Если a AND b = c ^ 4 то
        Результат = с / d / b + a
     Иначе
        Если c + b = a ^ 3 + b ^ 3 то
           Результат = a XOR (b + c)
        Иначе
           Результат = b >> 3 (сдвинуть с три раза вправо)

3) Если a * c + b * d = a * d + b * c то
        Результат = a ^ 2
     Иначе
        Если a > c то
           Результат = (с AND b)
        Иначе
           Результат = a – (b OR c)

4) Если a ^ 2 = b * c
        Если c * b = d / b то
           Результат = a OR b
        Иначе
           Результат = с
     Иначе
        Результат = с * a - b

5) Если a AND (a - 1) = b то
        Если d > c + b то
           Результат = c % d + c / d
        Иначе
           Результат = c XOR d
     Иначе
        Результат = b OR (b + 1)

6) Если a OR (a + 1) = b то
        Результат = (a * b + c) % d
     Иначе
        Если a AND b = c OR d то
           Результат = b AND (b - 1)
        Иначе
           Результат = c % d + b * a

7) Найти максимум между a*b, b*c, c*d, d*a.

8) Найти максимум между a, b, c, d и отнять от него минимум между этими числами.

9) Написать программу, которая помещает в регистр AX наибольшее из 4 чисел.

10) Написать программу, которая помещает в регистр AX наименьшее из 4 чисел.
